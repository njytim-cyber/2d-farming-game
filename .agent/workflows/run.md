# /run Workflow

## When to Use
- Starting development session
- Testing changes locally
- Playing the game

## Steps

1. **Install Dependencies** (first time only)
   // turbo
   ```bash
   npm install
   ```

2. **Start Development Server**
   // turbo
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   - Vite will auto-open http://localhost:3000
   - Or manually navigate to the URL

## Controls

### Keyboard
- **WASD** or **Arrow Keys**: Move
- **E** or **Enter** or **Space**: Interact

### Mouse/Touch
- **Click tile**: Move to or face tile
- **ACT button**: Interact with facing tile
- **+/-**: Zoom in/out

### Inventory
- **Click slot**: Select item

## Quick Tips
- Till grass (green) to create soil (brown)
- Plant seeds on soil tiles
- Stand next to crops and press ACT to harvest
- Visit the shop building to buy seeds
- Go home and sleep to restore energy
